# What is Array?
Array -> like a list of various things

<br>
This various things list is literally anything, because:

> <strong>Arrays can contain items of any datatype</strong>

# Methods
`push()` -> add item to array

`pop()` -> delete item

`shift()` -> delete 0-index value, decrement 1 to all items and return deleted value

`unshift()` -> add new item to array and return new array length

`splice()`-> syntax: `splice(start_index, amount_items_to_remove?, item_to_add_1, item_to_add_2, ...)`
