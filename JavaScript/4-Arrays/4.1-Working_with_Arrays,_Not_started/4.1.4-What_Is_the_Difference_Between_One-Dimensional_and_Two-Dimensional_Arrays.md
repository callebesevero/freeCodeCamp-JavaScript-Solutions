# Theory
## What the difference between one-dimensional array to two-dimensional array?

### One-dimensional arrays
One-dimensional arrays are like a single list.  
To acess each element in this "list", we parse the index associated to him.

Example: `list[0]` -> get the first item

### Two-dimensional arrays
Two-dimensional arrays are an array inside other array.  
Serves to represent more complex dataset, like chessboard, spreadsheets, etc. 

Example:
```
let chessboard = [
    ["R", "N", "B", "Q", "K", "B", "N", "R"],
    ["P", "P", "P", "P", "P", "P", "P", "P"],
    [" ", " ", " ", " ", " ", " ", " ", " "],
    [" ", " ", " ", " ", " ", " ", " ", " "],
    [" ", " ", " ", " ", " ", " ", " ", " "],
    [" ", " ", " ", " ", " ", " ", " ", " "],
    ["p", "p", "p", "p", "p", "p", "p", "p"],
    ["r", "n", "b", "q", "k", "b", "n", "r"]
];
```

To access the "upper" queen, we use `chessboard[0][3]` = `"Q"`.
