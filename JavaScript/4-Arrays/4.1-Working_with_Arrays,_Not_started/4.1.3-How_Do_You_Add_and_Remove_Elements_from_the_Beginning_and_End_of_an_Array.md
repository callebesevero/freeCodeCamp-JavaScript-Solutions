# Theory
## Methods
`push()` -> add item(s) to end / returns new array length

`pop()` -> remove item(s) from end / returns detelet item(s)

`shift()` -> remove item(s) from start / returns deleted item(s)

`unshift()` -> add item(s) to start / returns new array length
