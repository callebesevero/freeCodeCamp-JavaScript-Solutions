# Theory

## Array destructuring
Having an array of fruits:
```
const fruits = ["apple", "banana", "pineapple"];
```

To jump a value, we use commas:
```
let [start, , end] = fruits;

// start -> "apple"
// end -> "pineapple"
```

We can get each of there as if it were a variable:
```
let [start, mid, end] = fruits;

// start -> "apple"
// mid -> "banana"
// end -> "pineapple"
```

It's possible assign a value to array destructuring variable, like this:
```
let [start, mid, end, other = "blueberry"] = fruits;

// start -> "apple"
// mid -> "banana"
// end -> "pineapple"
// other -> "blueberry"
```

If we need to save the rest to the array, use:
```
let [start, ...rest] = fruits;

// start -> "apple"
// rest -> ["banana", "pineapple"]
```

It take away the need to initialize a single variable and parse the index related to the item.

### `...rest`
Yet is possible add the rest of a list to a variable, like in example:
```
let [first, ...rest] = fruits

// first -> "apple"
// rest -> ["banana", "pineapple"]
```

`...rest` must be exclusively in the end to the "destructuring list"
