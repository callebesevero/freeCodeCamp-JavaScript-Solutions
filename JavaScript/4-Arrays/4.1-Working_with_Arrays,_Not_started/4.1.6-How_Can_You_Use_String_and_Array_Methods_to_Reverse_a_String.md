# Theory
## Methods to reverse a string
`.split(arg)` -> divide the string by a separator (arg -> `""`, `" "`, `"-"`) 
`.reverse()` -> reverse the array
`.join(arg)` -> join the array to a unique string (arg -> separator between they)
