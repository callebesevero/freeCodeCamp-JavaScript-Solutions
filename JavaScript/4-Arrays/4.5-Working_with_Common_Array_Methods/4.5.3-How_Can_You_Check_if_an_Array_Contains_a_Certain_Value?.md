# Theory 

The method `includes()` allow to check if an element is in the array.

## Basic syntax

```
array.includes(elementToSearch, startIndex?)
```

- `elementToSearch`: element to check if it's in the array
- `startIndex`: index from which the search begins

Return `true` if contains or `false` if does not contain.
