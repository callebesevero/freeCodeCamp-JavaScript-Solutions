# Theory 

## Basic syntax 

```
array.splice(startIndex, itemsToRemove, item...)
```

- `startIndex`: start index
- `itemsToRemove`: quantity of items to remove
- `item`: item to add (added from the `startIndex`)
- `...`: more items can be added to `array`

Return an array with deleted elements.

## Other uses

### Add items in middle of the array

```
array.splice(middleIndex, 0, item...)
```

- `0`: none item are removed

### Clear the array

```
array.splice(0)
```

- When the second parameter is not passed (that limits how many items will be removed), all items in the right of the array are removed

### Inefficiency of this method 

Like this method shift the elements to the right, the more items there are after the start index, more work will be to reindex
