# Theory 

Shallow copies allow us to do operations and diverse things in an array, without modifying the original array.

There are 3 methods to do it:
- `concat()` 
- `slice()` 
- `...` (spread operator) 

### `concat()` 

Join an array with other array.

```
const originalArray = [1, 2, 3];
const copyArray = [].concat(originalArray);

console.log(originalArray === copyArray); // false
```

### `slice()` 

Slice the array from an index to other index.

```
const originalArray = [1, 2, 3];
const copyArray = [...originalArray];

console.log(originalArray === copyArray); // false
```

### `...` (spread operator) 

Spreads elements from an array to other array.

```
const originalArray = [1, 2, 3];
const copyArray = originalArray.slice();

console.log(originalArray === copyArray); // false
```
